Setting up your browser

This site requires few settings turned on or off in your browser.

It is:

  • Downloading files without prompting for confirmation
  • Pop-ups turned on (just for this domain)
  • Auto downloading mp3 fles (few modern browser starts playing mp3 song instead of downloading)
  • VLC plugin for browser turned off for mp3 files

You can set it up in every browser but I recomend do it in Firefox v.10 (it's really old version, I know but it's good because it has no plugins for mp3). You can download it here.

After the instalation, we still need to set up few thinks in browser settings, so open browser and its settings.

On the first tab called "General" set up your destination folder - I recommend new, empty folder in your music library. You can also untick "show the download window...", but if you do that you'll have to open this window manually everytime you'll want to check download state.

Now go on to tab "Content" and add this domain to exceptions in pop-ups (or just completly turn on pop-ups).

In tab "applications" find everything connected with mp3 files - eq. "MPEG layer 3 (audio/mpeg)" or "MPEG layer 3 (audio/x-mpeg)" and choose the option "save file".

At the end turn off auto updating your browser - tab "advanced", tab "update".

Now you're done. Now click here to test your settings - if file start downloading, everything is set up correct.
